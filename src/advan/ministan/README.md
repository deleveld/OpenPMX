# ministan: a modernized version of Stanpump

The files in this directory come from `ministan`, a modernized version of 
Stanpump. It is available at https://framagit.org/jaj/ministan

The code in this directory was developed by Jona Joachim (jona@joachim.cc) and published in:

 - Joachim J. Deep dive into STANPUMP [Internet]. 2025. Available from: https://jaj42.github.io/blog/posts/202508_stanpump/index.html
 - Joachim, Jona. 2026. “Unit Disposition Functions and the TCI Algorithm.” February 11, 2026. https://jaj42.github.io/blog/posts/202602_udf/index.html. 

The code here was derived from the original Stanpump code which was developed by Steve Shafer and
is avalable at: https://opentci.org/code/stanpump

Also credited are:

 - Steven Shafer for creating STANPUMP and releasing it as open source
